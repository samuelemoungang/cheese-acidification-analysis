# Cheese Acidification Analysis — Project Overview

## Objective
Determine **cheese quality** as a function of **pH**, **temperature**, and **time** profiles during fermentation. According to the supervising professor (Dimitri Bocquel), this approach — modelling cheese quality from acidification kinetics — **has never been done in this form before**.

## Team
- **Supervisor / Professor**: Dimitri Bocquel
- **Authors / Experimenters**: Lilandra Albert-Lavault & Samuele Moungang

## Instrumentation
| Instrument | Type | Measurements |
|---|---|---|
| USB logger (brand unspecified) | pH meter | pH + mV, 1 min |
| Server logger | pH meter | pH, ~1 s |
| Hanna Instruments edge (PHLOT001, PHLOT002) | pH meter | pH + mV + T°C, 1 min |
| Elpro Ecolog TP4-L (ID 409302) | Temperature logger | T°C, 16 s |

## Session 1 — 28 April 2026
- **Milk used**: 282 kg
- **Cheese produced**: 34.34 kg → **~5.7 kg per piece**
- **Fresh yield**: **12.17%**  (34.34 / 282)
- Active probes: USB (Fromage 1), Server (Fromage 2), Hanna PHLOT001 (Fromage 3), Elpro temperature logger

## Session 2 — 5 May 2026
- **Cheese after unmolding**: 33.26 kg
- **Fresh yield**: **11.8%**
- Active probes: Hanna PHLOT002 (single pH meter), Elpro temperature logger

## Quality Variable
**Fresh yield** (kg cheese / kg milk × 100) is the only quantitative quality indicator currently available. Session 1: 12.17%, Session 2: 11.8%.
